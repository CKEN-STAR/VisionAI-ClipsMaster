> **从入门到精通的完整教程 - AI驱动短剧混剪工具**

## 🎬 核心功能使用

### 1. AI剧本重构功能

#### 步骤1：导入原始字幕
```
1. 点击"导入原片SRT"按钮
2. 选择您的原始字幕文件 (.srt格式)
3. 系统自动检测语言 (中文/英文)
4. 显示导入成功提示
```

#### 步骤2：AI智能重构
```
1. 点击"开始AI重构"按钮
2. AI分析原始剧本结构
3. 自动生成爆款短剧版本
4. 实时显示重构进度
```

#### 步骤3：预览重构结果
```
- 查看重构后的字幕内容
- 对比原版与爆款版差异
- 调整重构参数 (可选)
- 确认重构效果
```

### 2. 视频片段匹配

#### 自动时间轴对齐
```
1. 导入对应的视频文件
2. 系统自动分析时间轴
3. 匹配字幕与视频片段
4. 生成精确的时间映射
```

#### 手动调整 (高级功能)
```
- 微调时间轴偏移
- 调整片段边界
- 设置淡入淡出效果
- 优化同步精度
```

### 3. 剪映项目导出

#### 导出设置
```
1. 选择导出格式: 剪映专业版
2. 设置输出目录
3. 配置项目参数
4. 点击"导出项目"
```

#### 导出结果
```
- 生成 .json 项目文件
- 包含多个独立视频片段
- 保持原始视频质量
- 支持剪映直接导入
```

## 🧠 AI模型管理

### 模型下载与配置

#### 自动推荐下载
```
1. 切换到"模型训练"标签页
2. 点击"智能推荐下载"
3. 系统检测硬件配置
4. 推荐最适合的模型版本
```

#### 支持的模型
```
中文模型:
- Qwen2.5-7B-Instruct-Q5 (推荐)
- Qwen2.5-7B-Instruct-Q4 (低内存)
- Qwen2.5-7B-Instruct-Q8 (高质量)

英文模型:
- Mistral-7B-Instruct-Q5 (推荐)
- Mistral-7B-Instruct-Q4 (低内存)
- Mistral-7B-Instruct-Q8 (高质量)
```

## 🚨 故障排除

### 常见问题解决

#### 启动问题
```
问题: 启动时间过长
解决: 
1. 使用 optimized_quick_launcher.py
2. 检查系统资源占用
3. 清理临时文件
4. 重启应用

问题: 启动失败
解决:
1. 检查Python版本 (需要3.8+)
2. 验证依赖安装
3. 查看错误日志
4. 重新安装依赖
```

#### 功能问题
```
问题: AI重构失败
解决:
1. 检查模型是否正确加载
2. 验证输入文件格式
3. 确认网络连接
4. 重新下载模型

问题: 导出失败
解决:
1. 检查输出目录权限
2. 确认磁盘空间充足
3. 验证FFmpeg安装
4. 查看详细错误信息
```

#### 性能问题
```
问题: 内存使用过高
解决:
1. 启用低内存模式
2. 关闭其他应用程序
3. 使用较小的模型
4. 重启应用释放内存

问题: 处理速度慢
解决:
1. 检查CPU使用率
2. 优化系统设置
3. 使用更快的存储设备
4. 升级硬件配置
```
