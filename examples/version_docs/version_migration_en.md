## Version Migration Guide

### 3.1 → 3.0 (Downgrading)

* **Risk Level:** High
* **Potential Information Loss:** Yes
* **Recommended Migration Path:** Incompatible, requires recreation

#### Migration Steps

1. 备份您的原始项目文件
2. 创建新的 3.0 版本项目
3. 手动重新创建项目内容

#### Important Notes

All features are compatible
