- 内存仪表盘截图
- 预警系统截图
- 用户界面示例图
- 其他文档相关图片

### 预警系统

## 使用说明

在文档中引用这些图片时，使用相对路径，例如：

```markdown
![内存仪表盘概览](../resources/images/memory_dashboard_overview.png)
```

## 注意事项

- 保持图片命名规范一致
- 建议使用PNG格式，保证清晰度
- 避免上传过大的图片文件（建议控制在1MB以内）
