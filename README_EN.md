## Project Overview

VisionAI-ClipsMaster is a revolutionary video processing tool designed specifically for short drama remix creators. It leverages locally deployed large language models (Qwen2.5-7B and Mistral-7B) to intelligently analyze original video plots, restructure subtitles, and generate engaging viral short video content.

The core advantage of this tool is **fully local processing** without relying on cloud services. It runs efficiently even on lower-spec computers with 4GB RAM and no dedicated GPU.

## Core Features

- **Intelligent Screenplay Restructuring**: Analyzes original subtitles using large language models
- **Bilingual Model Support**: Integrates Qwen2.5-7B (Chinese) and Mistral-7B (English)
- **Lightweight Design**: Works on computers with 4GB RAM/no dedicated GPU
- **Model Training Functionality**: Provides a training interface for learning viral patterns
- **Zero Video Processing**: Pure splicing strategy without complex editing
